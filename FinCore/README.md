# FinCore

O **FinCore** é uma solução de gestão financeira pessoal desenvolvida para oferecer uma visão clara, organizada e inteligente sobre suas receitas, despesas, investimentos e dívidas.

---

## 🛠️ Tecnologias Utilizadas

### Back-end
* [cite_start]**Framework:** FastAPI (Python) [cite: 1]
* [cite_start]**ORM:** SQLAlchemy [cite: 1]
* [cite_start]**Autenticação:** JWT (python-jose) e bcrypt [cite: 1]
* [cite_start]**Processamento de Dados:** Pandas e OpenPyXL (para exportação em Excel) [cite: 1]

### Front-end
* [cite_start]**Framework:** React [cite: 1]
* [cite_start]**Bundler:** Vite [cite: 1]
* [cite_start]**Roteamento:** React Router DOM [cite: 1]
* [cite_start]**Comunicação:** Axios [cite: 1]
* [cite_start]**Visualização:** Chart.js e Recharts [cite: 1]
* [cite_start]**Ícones:** Lucide React [cite: 1]
